# Practical 1 - TikTok Clone

## What is this project?
This ia a simple practical to clone TikTok Website. 
I used next.js and Tailwind CSS.

## What I built
- A video feed on home page
- Navigation links on the sidebar
- Following, Explore, Live, Upload and Profile pages
- Login and signup pages with error message production.

## Tools and Technologies Used
- **Next.js** - For web application building
- **Tailwind CSS** - For styling
- **React Icons** - For sidebar icons
- **React Hook Form** - To manage login and signup forms

## How to Run the Project

### Step 1: Clone the repository
```bash
git clone https://github.com/UgayNobu/SS2026_WEB101_02240369.git
```

### Step 2: Go to the project folder
```bash
cd Practical1_Tiktok/tiktok-clone
```

### Step 3: Install dependencies
```bash
npm install
```

### Step 4: Run the development server
```bash
npm run dev
```

### Step 5: Open in browser
```
http://localhost:3000
```
